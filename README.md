# Bloog Application

- This application will be built out on the MERN stack.
- Users will be able to cycle through different posts and choose what is most appealing to them.
- Users can also create posts.
- If a post receives more dislikes than likes, the post then gets removed from the feed forever.

# Technologies Used

- Typescript, TailwindCSS, Axios, DaisyUI, ReactJS, Express, NodeJS, MongoDB
- More to be added soon.
